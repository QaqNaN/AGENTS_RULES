# AGENTS.md Template for Code Agents

这是一个用于约束 code agent 行为的 `AGENTS.md` 模板，主要面向 Codex、Claude Code、Cursor Agent 等代码辅助工具。

在实际使用 code agent 开发个人项目的过程中，我发现如果没有明确的规则，agent 很容易出现以下问题：

- 擅自重构
- 修改无关代码
- 引入不必要的抽象
- 静默吞掉错误
- 跳过验证步骤
- 声称测试通过但实际没有运行
- 为了让问题消失而削弱类型检查或 lint 规则
- 表面上遵循最小改动，实际却采用打补丁式的绕路修改

因此，我整理了这份 `AGENTS.md`，用于给 agent 提供更清晰的工作边界。

## 目标

这个模板的核心目标是：

- 让 agent 默认采用最小必要改动，同时优先选择直接、清晰、局部的修改方式
- 优先复用现有接口、工具函数、组件和代码风格
- 在需求不明确时先讨论方案，而不是直接改代码
- 禁止擅自进行大规模重构
- 禁止静默处理错误
- 禁止伪造测试、lint、构建或手动验证结果
- 修改完成后明确说明改了什么、验证了什么、还有什么风险
